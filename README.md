### Install
```
npm install frappe-gantt-with-roles or yarn add frappe-gantt-with-roles
```

### Usage
Include it in your HTML:
```
<script src="frappe-gantt.min.js"></script>
<link rel="stylesheet" href="frappe-gantt.css">
```
